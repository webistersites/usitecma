=== About Legal ===
Legal is a modern looking premium quality WordPress theme. It is clean, super flexible , fast in loading and bootstrap based responsive theme for business or personal websites. Legal Theme was developed specially for Advocates or Attorneys in mind. But this can be used for any business. It has got custom widgets, theme-options etc in order to get it customized as per your needs.

== Installation ==

1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "legal.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (legal.zip) that you have downloaded.
 = Upload the entire folder (legal) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/legal/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "legal" to activate the theme.

5. Theme Options :

	a) Basic Settings:
           
           You can manage here site logo,favicon, and contact information text.	

	b) Home Page Settings:

		i) Banner Slider  : here you can add Slider image and link.
		ii) Title Bar : You can set home page title and description.
                iii) Benefits Section : you can insert font-awesome icon class, title and short description. 
                iv) Background Section : you can upload background image, title and link. 
                v) Recent post Section : you can select category of posts and add title of posts slider.
                
        c) Social Settings:

            You can add here social media links.
        
        d) Footer Settings:

            you can add copyright text from here.
